Git- распределенная система контроля версий 

Это система для отслеживания и ведения истории изменения файлов в вашем проектеG
Репозиторий- хранилище вашего кода и историии его изменений 
GitHub - Веб сайт для размещения git-репозиториев в совместной разработке пректов

Команды GIT:

1.git init-она создает новый локальный гит репозиторий у вас на пк в той директории в которой была прописана команда (Команда вводится один раз и в начале)

2. git add - команда для добавления файлов в систему отслеживания гита после этой команды изменения начинают отслеживаться гитом

3.-git commit -m '<comemnt>' - как только достигаем определенного момнта в разработке то тогда мы сохраняем и комментируем изменения
 в локальный репозиторий 


4. Git remote add - это команда для того чтобы связать наш локальный репозиторий с удаленным репозиторием то есть добавляет подключение к репозиториям на гитхабе

git remote add <ssh>NAME ORIGIN> <ССЫЛКА_на peno:>

5. Git push - команда для отправки всех изменений на локальном репозитории в удаленный репозиторий 
  
   Git push origin <branch(main)>
